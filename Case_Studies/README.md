# Case Study: Customer Complaint Triage System (Auto-Routing)

## **Summary**

Design an NLP system that automatically routes short, noisy customer complaints to the correct department. Key challenges: short text, spelling errors, slang, sarcasm, and mixed intents. The solution combines robust preprocessing, transformer-based classification, and intent segmentation.
